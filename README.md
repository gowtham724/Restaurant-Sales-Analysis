# Restaurant Sales Analysis
Data science
This project focuses on analyzing restaurant sales data to gain meaningful insights and improve decision-making using machine learning techniques. Restaurants generate large volumes of transactional data every day, including information related to item prices, order quantities, discounts, taxes, total bill amounts, and customer types. However, this data is often underutilized, leading to inefficient sales strategies and missed revenue opportunities.

In this project, a structured restaurant sales dataset was used to preprocess and analyze historical transaction data. Data cleaning techniques such as handling missing values and removing duplicates were applied, followed by feature encoding and feature scaling to prepare the dataset for modeling. The dataset was then split into training and testing sets to evaluate model performance accurately.

A classification model was developed to predict high-sales and low-sales orders based on various input features. Advanced machine learning algorithms such as Random Forest and Logistic Regression were used to improve predictive performance. By applying appropriate preprocessing techniques and selecting an optimal model, the system achieved approximately 80% accuracy.

The results of this project help restaurant management understand customer purchasing behavior, identify high-performing sales patterns, and optimize pricing and promotional strategies. Overall, the project demonstrates how data-driven approaches can enhance operational efficiency and profitability in the restaurant industry.
